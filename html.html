O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para a criação de páginas web. Ele descreve a estrutura de uma página web utilizando uma série de elementos e tags.

Como Funciona o HTML?

O HTML utiliza tags para definir elementos dentro de uma página. Essas tags são delimitadas por colchetes angulares e geralmente vêm em pares, com uma tag de abertura e uma tag de fechamento:

<elemento>Conteúdo</elemento>

Principais Elementos HTML

Aqui estão alguns dos elementos mais comuns do HTML:

  • <h1> a <h6>: Definem cabeçalhos, com <h1> sendo o mais importante e <h6> o menos importante.
  • <p>: Define um parágrafo.
  • <a>: Define um hiperlink.
  • <img>: Define uma imagem.
  • <div>: Define uma divisão ou seção de um documento.
  • <span>: Define uma seção em linha dentro de um documento.
