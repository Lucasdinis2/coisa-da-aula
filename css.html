O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para controlar a apresentação de documentos HTML. Com o CSS, é possível definir cores, fontes, margens, espaçamentos e outros aspectos visuais de uma página web.

Como Funciona o CSS?

O CSS funciona através de regras de estilo aplicadas a elementos HTML. Cada regra é composta por um seletor e um bloco de declarações:

seletor {
  propriedade: valor;
}

Principais Propriedades CSS

Aqui estão algumas das propriedades mais comuns do CSS:

  • color: Define a cor do texto.
  • background-color: Define a cor de fundo de um elemento.
  • font-family: Define a fonte do texto.
  • font-size: Define o tamanho da fonte.
  • margin: Define a margem externa de um elemento.
  • padding: Define o preenchimento interno de um elemento.
  • border: Define a borda de um elemento.
  • width e height: Definem a largura e a altura de um elemento.
